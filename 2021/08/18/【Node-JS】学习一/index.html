【Node.JS】学习一

一、Node.js介绍

1.1 Node.js是什么

Node.js is a JavaScript runtime bult on Chrome’s v8 JavaScript engine.

Node.js 不是一门语言

Node.js 不是库、不是框架

Node.js是一个JavaScript运行时环境，可以解析和执行JavaScript 代码，以前只有浏览器可以。

浏览器中的JavaScript

EcmaScript

BOM

DOM

Node.js 中的JavaScript

没有BOM、DOMo EcmaScript

在Node这个JavaScript 执行环境中为JavaScript提供了一些服务器级别的操作API，例如文件读写。

网络服务的构建

网络通信

http服务器等处理。。。

（1）核心模块

Node为JavaScript 提供了很多服务器级别的API，这些API绝大多数都被包装到了一个具名的核心模块中了。

例如文件操作的fs核心模块，http服务构建的http模块，path路径操作模块、os 操作系统信息模块。。。。

（2）模块系统

require是一个方法

它的作用就是用来加载模块的

在Node中,模块有三种:

1、具名的核心模块,例如fs、 http

2、用户自己编写的文件模块

3、第三方模块，需要npm安装后使用

Node中没有全局作用域，只有模块作用域

require方法有两个作用:

1．加载文件模块并执行里面的代码

2．拿到被加载文件模块导出的接口对象

在每个文件模块中都提供了一个对象：exexports

exexports默认是一个空对象

可以把所有需要被外部访问的成员挂载到这个对象上

Node.js uses an event-driven, non-blocking l/o model that makes it lightweight and efficient.

event-driven事件驱动

non-blocking l/o model非阻塞lO模型〔异步)

llightweight and efficient.轻量和高效

Node.js package ecosystem, npm, is the largest ecosystem of open source libraries in the world.

包管理工具

npm是世界上最大的开源库生态系统

绝大条数JavaScript相关的包都存放在了npm上，这样做的目的是为了让开发人员更方便的去下载使用。
